Detailed Description

Image vector table structure.

This structure defines the layout of the Image Vector Table (IVT) used by the i.MX RT1060 ROM bootloader. The IVT contains critical information for the bootloader, including the entry point for the application, pointers to configuration data, and other metadata required for the boot process.

Note
The fields of this structure are defined according to the i.MX RT1060 reference manual, section 9.7.1.1
