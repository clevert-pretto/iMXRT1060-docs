Detailed Description

Boot data structure.

This structure defines the layout of the Boot Data Table used by the i.MX RT1060 ROM bootloader. The Boot Data Table contains metadata about the application image, including its physical location and size constraints. This information is critical for the bootloader to correctly load and execute the application.

Note
The fields of this structure are defined according to the i.MX RT1060 reference manual, section 9.7.1.2
