Detailed Description

Configuration structure for FlexSPI NOR flash devices.

This structure is used by the i.MX RT1060 ROM bootloader. This structure must be placed at a specific memory location defined in the linker script to be correctly recognized by the bootloader. It contains all necessary parameters for initializing and configuring the FlexSPI interface and the attached NOR flash device during the boot process.

Note
The fields of this structure are defined according to the i.MX RT1060 reference manual, section 9.6.3.1
