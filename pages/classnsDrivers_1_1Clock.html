Detailed Description

Core Clock configuration driver class.

This class provides functions to configure and control the core clock of the i.MX RT1060 microcontroller. It includes functions to set the core frequency, configure PLLs, configure clock dividers, and configure clock multiplexers.

Note
This class should be used to configure the core clock of the i.MX RT1060 microcontroller.
